# capybara-demo

## 简介
对于web开发，有些测试是属于重复性的测试，重复性的测试就可以自动化，这里写了一下自动化测试事例。主要支持`localStorage`

具体文档  
<https://github.com/jnicklas/capybara>

## 使用方式

### 1. 安装

```shell
git clone https://github.com/watsy0007/capybara-demo.git ./capybara-demo
cd capybara-demo
bundle install
```

### 2. 测试

```shell
rspec spec/login_spec.rb
```
